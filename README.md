# SomeeStyle Professional Pencil Sketch Converter

Transform photos into realistic pencil sketches instantly with professional-grade image processing algorithms.

## Features

✅ **99% Facial Resemblance** - Perfect face preservation  
✅ **Pure Black & White** - Always monochrome output  
✅ **4 Quality Presets** - Light, Balanced, Detailed, Ultra  
✅ **Advanced Algorithms** - NPR (Non-Photorealistic Rendering) techniques  
✅ **Instant Results** - 1-2 second processing  
✅ **Free Forever** - No API costs, runs in browser  
✅ **Offline Capable** - No internet required after loading

## Technology

- Bilateral filtering for edge preservation
- Adaptive histogram equalization (CLAHE)
- Multi-scale edge detection (Sobel + Laplacian)
- Dodge & burn blending
- Directional hatching simulation
- Realistic pencil grain with Perlin-like noise
- Local contrast enhancement (unsharp mask)
- Professional tone mapping
